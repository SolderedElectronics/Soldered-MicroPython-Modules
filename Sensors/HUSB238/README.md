# How to install

After [**installing the mpremote package**](https://docs.micropython.org/en/latest/reference/mpremote.html):

    mpremote mip install github:SolderedElectronics/Soldered-MicroPython-Modules/Sensors/HUSB238

Or Windows:

    python -m mpremote mip install github:SolderedElectronics/Soldered-MicroPython-Modules/Sensors/HUSB238

# How to install

---

After [**installing the mpremote package**](https://docs.micropython.org/en/latest/reference/mpremote.html):

```sh
  mpremote mip install github:SolderedElectronics/Soldered-MicroPython-Modules/Sensors/SimpleSensor
```

Or Windows:

```sh
  python -m mpremote mip install github:SolderedElectronics/Soldered-MicroPython-Modules/Sensors/SimpleSensor
```

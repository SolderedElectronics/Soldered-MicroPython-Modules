# How to install

After [**installing the mpremote package**](https://docs.micropython.org/en/latest/reference/packages.html):

    mpremote mip install github:SolderedElectronics/Soldered-MicroPython-Modules/Sensors/L86M33

Or Windows:

    python -m mpremote mip install github:SolderedElectronics/Soldered-MicroPython-Modules/Sensors/L86M33

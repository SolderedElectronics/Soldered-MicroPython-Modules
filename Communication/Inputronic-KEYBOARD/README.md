# How to install

------------------------------------------------------------------------

After **installing the mpremote package**
(https://docs.micropython.org/en/latest/reference/mpremote.html), flash
the Inputronic Keyboard module to the board using the following command:

``` sh
mpremote mip install github:SolderedElectronics/Soldered-Micropython-modules/Communication/Inputronic-KEYBOARD
```

Or, if you're running a Windows OS:

``` sh
python -m mpremote mip install github:SolderedElectronics/Soldered-Micropython-modules/Communication/Inputronic-KEYBOARD
```

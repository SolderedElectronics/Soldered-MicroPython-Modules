# How to install

---

After [**installing the mpremote package**](https://docs.micropython.org/en/latest/reference/mpremote.html), flash a module to the board using the following command:

```sh
  mpremote mip install github:SolderedElectronics/Soldered-MicroPython-Modules/Actuators/mcp4018
```
Or, if you're running a Windows OS:

```sh
  python -m mpremote mip install github:SolderedElectronics/Soldered-MicroPython-Modules/Actuators/mcp4018
```
